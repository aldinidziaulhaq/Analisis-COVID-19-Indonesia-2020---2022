## Analisis COVID-19 Indonesia 2020 - 2022
Dashboard interaktif untuk menganalisis tren kasus, kematian, 
dan kesembuhan COVID-19 di Indonesia dari tahun 2020 hingga 2022.

## Deskripsi
Project ini merupakan analisis data COVID-19 Indonesia menggunakan 
Python dan Streamlit.

## Sumber Data
Dataset: COVID-19 Indonesia
Sumber: Kaggle
